# CLI Utility Tool (Node.js)

A simple Command Line Interface (CLI) application built using **Node.js**, **Commander.js**, and **Axios**.  
This tool allows users to perform basic calculations, greet users, and fetch data from public APIs such as jokes, quotes, and Pokémon information.

---

## Features

- Greeting command
- Basic calculator operations
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Fetch a random joke
- Fetch a motivational quote
- Fetch Pokémon information
- Custom commands

---

## Technologies Used

- Node.js
- Commander.js
- Axios
- Public APIs
